# Keyboard Shortcuts

All shortcuts work when the solar system view is focused.

## Navigation

| Key | Action |
|-----|--------|
| `N` | Create new task |
| `Escape` | Close open panels |
| `H` | Hide / show all UI overlays |
| `R` | Reset camera to default position |

## Display Toggles

| Key | Action |
|-----|--------|
| `L` | Toggle planet labels |
| `M` | Toggle moon labels |
| `O` | Toggle orbit ring lines |

## Animation

| Key | Action |
|-----|--------|
| `Space` | Pause / resume orbital animation |

---

> **Tip:** Hold `Shift` + `Left drag` to pan the camera.
> Scroll to zoom in / out. Right-drag to rotate.

---

## Mouse Controls

| Action | Result |
|--------|--------|
| Left drag | Rotate / orbit camera |
| Scroll wheel | Zoom in / out |
| Click planet | Open task details |
| Right click | Context menu |
